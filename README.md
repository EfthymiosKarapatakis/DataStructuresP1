# Project για το μάθημα Δομές Δεδομένων στην σχολή CEID του Πανεπιστημίου Πατρών

## Εκφώνηση:

Σας ζητείται να υλοποιήσετε προγράμματα σε γλώσσα της επιλογής σας (κατά προτίμηση C αν και Java, C++ και Python είναι επίσης αποδεκτές), που να χρησιμοποιούν ως είσοδο δεδομένα της επιλογής σας και θα υλοποιούν τις παρακάτω λειτουργίες:

### 1. (Ελεγκτής έκφρασης με στοίβα)

Υλοποιήστε συνάρτηση που χρησιμοποιεί στοίβα χαρακτήρων για να ελέγξει αν μια έκφραση περιέχει σωστά τοποθετημένες παρενθέσεις `()`, αγκύλες `[]` και άγκιστρα `{}`.

Η συνάρτηση πρέπει να επιστρέφει:
- `1` αν η έκφραση είναι σωστή
- `0` διαφορετικά

Τεστάρετε τη συνάρτηση με ένα κυρίως πρόγραμμα με δεδομένα της επιλογής σας και δοκιμάστε ενδεχομένως και διαφορετικές υλοποιήσεις (λίστα ή πίνακας για τη στοίβα που θα χρησιμοποιήσετε).

---

### 2. (Αναζήτηση/Ένθεση/Διαγραφή τιμής σε απλό (μη ζυγισμένο) δυαδικό δέντρο)

Υλοποιήστε συνάρτηση με το όνομα `binaryTreeSearch` που επιχειρεί να εντοπίσει μια καθορισμένη τιμή σε ένα δυαδικό δέντρο αναζήτησης με ακεραίους σαν περιεχόμενο.

Η συνάρτηση θα πρέπει:
- να λαμβάνει ως ορίσματα έναν δείκτη στην ρίζα του δυαδικού δέντρου
- και ένα κλειδί αναζήτησης

Αποτελέσματα:
- Επιστρέφει δείκτη στον κόμβο αν βρεθεί
- Διαφορετικά επιστρέφει `NULL`

Επιπλέον:
- Υλοποιήστε συναρτήσεις ένθεσης και διαγραφής τιμής από το δέντρο
- Τεστάρετε με κύριο πρόγραμμα

Υλοποιήστε είτε:
- φυλλοπροσανατολισμένο
- είτε κομβοπροσανατολισμένο δέντρο

Εξηγήστε θεωρητικά:
- τη διαφοροποίηση των δύο επιλογών
- πώς θα άλλαζε ο κώδικας

---

### 3.

#### (i) (Διέλευση δυαδικού δέντρου κατά επίπεδα)

Η διέλευση αυτή εμφανίζει τις τιμές των κόμβων:
- από επίπεδο προς επίπεδο
- από αριστερά προς τα δεξιά
- ξεκινώντας από τη ρίζα

Ο αλγόριθμος:
1. Εισάγετε τη ρίζα στην ουρά
2. Ενόσω υπάρχουν κόμβοι στην ουρά:
   - Πάρε τον επόμενο κόμβο
   - Εκτύπωσε την τιμή του
   - Αν υπάρχει αριστερό παιδί → εισαγωγή στην ουρά
   - Αν υπάρχει δεξί παιδί → εισαγωγή στην ουρά

#### (ii) (Διελεύσεις δυαδικού δέντρου)

Υλοποιήστε:
- Προδιάταξη (Preorder)
- Μεταδιάταξη (Postorder)

Τεστάρετε με κύριο πρόγραμμα.

---

### 4.

#### (Ύψος δυαδικού δέντρου)
Υλοποιήστε συνάρτηση που:
- λαμβάνει δείκτη στη ρίζα
- επιστρέφει το ύψος του δέντρου

#### (Βάθος κόμβων)
Υλοποιήστε συνάρτηση που:
- λαμβάνει δείκτη στη ρίζα
- υπολογίζει και εκτυπώνει το βάθος κάθε κόμβου

Τεστάρετε και τις δύο με κύριο πρόγραμμα.

---

### 5. (Ανάπλαση από διαπεράσεις δυαδικού δένδρου)

Εξηγήστε πώς μπορούμε να αναπλάσουμε ένα δυαδικό δέντρο όταν έχουμε:

- Ενδοδιάταξη (Inorder)
- και Μεταδιάταξη (Postorder)

Υλοποιήστε πρόγραμμα που:
- διαβάζει τις δύο διαπεράσεις
- τις αποθηκεύει ως συμβολοσειρές

Υποθέσεις:
- Οι κόμβοι είναι μοναδικοί χαρακτήρες
- Η ανάπλαση σημαίνει εύρεση αριστερού και δεξιού παιδιού για κάθε κόμβο

Επαναλάβετε:
- χρησιμοποιώντας Προδιάταξη (Preorder) αντί για Μεταδιάταξη

Επιβεβαίωση:
- μέσω των συναρτήσεων διαπέρασης του ερωτήματος 3
